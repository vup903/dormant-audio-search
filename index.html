Ask a question of 1930s–1950s radio news. Every answer points back to the original recording.

This demo runs the full two-stage retrieval pipeline — hand-built BM25, MiniLM semantic search, and cross-encoder reranking — entirely in your browser (ONNX via transformers.js). No server, no API. First visit downloads the models (~50 MB, cached afterwards). Audio streams from archive.org.
Try: news about the invasion of Normandy presidential election campaign workers on strike atomic weapons

All audio is public-domain historical broadcast material streamed from archive.org — no NPR content. Transcripts and topic tags are AI-generated (faster-whisper + zero-shot tagging against a 20-term controlled vocabulary) and labeled as such; expect transcription errors on 80-year-old recordings. Results come only from the indexed corpus — no index entry, no answer. AI suggests — a human decides.
Retrieval: semantic ∪ lexical top-8 → cross-encoder → top-5, one window per broadcast — the same code path as the Python app, ported line-for-line and scored at the same hit@5 on the golden query set.
